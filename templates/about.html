Dataset Overview

The dataset contains 16 anonymized numerical features named feature_0 to feature_15. These features represent customer attributes and behaviors relevant to predicting insurance churn.

How to Use This Application

  1. Upload your customer data CSV file on the Upload & Predict page. The file should include the anonymized features for each customer.
  2. Download the generated predictions once the processing completes. You can also explore the results interactively via the Dashboard page.
  3. Leverage the automated recommendations derived from churn probability scores to design and customize your customer retention campaigns, targeting at-risk customers effectively.

Purpose of the Tool

This tool uses a machine learning model to estimate the likelihood of churn for each customer based on their feature data. Understanding these risks enables insurance companies to take proactive actions to retain valuable customers and reduce churn rates.
